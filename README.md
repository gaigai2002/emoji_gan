# ✨ EMOJI-GAN：AI Emoji 生成器

[](https://github.com/gaigai2002/emoji_gan)

欢迎来到 EMOJI-GAN！这是一个使用**生成对抗网络 (GAN)** 来学习并生成全新 Emoji 表情的项目。


## 🛠️ 技术栈

本项目主要基于以下技术：

  * **Python 3.10**
  * **PyTorch** 
  * **GAN 模型** (DCGAN)
  * **NumPy** (用于数据处理)
  * **Matplotlib / PIL** (用于图像处理和可视化)


## 🎨 数据集


  * **项目使用了Kaggle的Full Emoji Image Dataset数据集**
